# AI Automation Agency Demo

A GitHub-ready front-end demo for an **AI Automation Agency for Local Businesses**.

This project gives you something real to show prospects. It demonstrates a simple automation use case:

**Missed Call / Lead Trigger → Instant Follow-Up → CRM Lead Creation**

## What is included

- `index.html` — dashboard view with KPIs, charts, and automation log
- `bookings.html` — interactive demo page where you trigger the workflow
- `workflow.html` — launch roadmap / kanban board
- `style.css` — shared styling
- `app.js` — demo logic, localStorage, charts, and CRM lead simulation
- `kanban.js` — drag-and-drop task board logic

## Why this is a good first demo

You can show a local business owner:
- how automation saves time
- how missed leads get captured faster
- how AI follow-up can improve booking chances
- how a dashboard can make the value visible

## How to run locally

### Option 1: easiest
Just open `index.html` in your browser.

### Option 2: use VS Code Live Server
1. Open the folder in VS Code
2. Install the **Live Server** extension
3. Right-click `index.html`
4. Click **Open with Live Server**

## How to put it on GitHub

1. Create a new GitHub repository
2. Upload all project files to the repo root
3. Commit the files
4. Go to **Settings > Pages**
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save
7. GitHub will generate a public link for your demo

- change it to missed calls, appointment reminders, rebooking texts
